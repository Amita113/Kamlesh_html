The transition Properties Specified One by One

Hover over the div element below, to see the transition effect:

short hand notation transition: width 2s linear 1s;

Note: The transition effect has a 1 second delay before starting.

Property Description transition A shorthand property for setting the four transition properties into a single property transition-delay Specifies a delay (in seconds) for the transition effect transition-duration Specifies how many seconds or milliseconds a transition effect takes to complete transition-property Specifies the name of the CSS property the transition effect is for transition-timing-function Specifies the speed curve of the transition effect
